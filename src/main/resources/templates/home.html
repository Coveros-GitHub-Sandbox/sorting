Sorting Example

This app just sorts a long list of words using various algorithms.

  • Unsorted
  • Bubble sort - O(n^2)
  • Insertion sort - O(n*2)
  • Merge sort - O(n log n)
  • Heap sort - O(n log n)
  • Quick sort - O(n log n)
  • Java sort - O(n log n)

If you want to see how these sorting methods work, you can look at some animations.
